# 新增运动健康组件

本次更新新增了4个优先级高的运动健康类组件，进一步完善了组件库的功能。

## 新增组件列表

### 1. StrengthTrainingChart - 力量训练记录

**功能特点：**
- 📊 记录多个训练动作的组数、重量、次数
- ✅ 支持完成状态标记
- 📈 自动计算总训练量、总次数
- 💪 显示肌肉群分类
- ⏱️ 休息时间提示
- 🔥 训练时长和消耗热量统计

**使用场景：**
- 健身房力量训练记录
- 私教课程数据追踪
- 个人训练计划管理

**示例路径：** `/strength-training-chart`

---

### 2. WorkoutZoneChart - 运动强度区间图

**功能特点：**
- 🎯 心率区间可视化（热身、燃脂、有氧、无氧、极限）
- 📊 区间时长和百分比统计
- 📈 ECharts 柱状图展示
- 💡 智能训练建议
- 🎨 区间颜色自动分级

**使用场景：**
- 专业运动员训练分析
- 心率监测数据展示
- 运动强度评估

**示例路径：** `/workout-zone-chart`

---

### 3. AchievementBadge - 成就徽章

**功能特点：**
- 🏆 多种稀有度（普通、稀有、史诗、传说）
- ⭐ 等级系统支持
- 🔒 未解锁状态和进度环
- ✨ 解锁动画效果
- 💫 发光特效
- 📏 三种尺寸（小、中、大）

**使用场景：**
- 运动成就系统
- 用户激励机制
- 里程碑展示

**示例路径：** `/achievement-badge`

---

### 4. ChallengeCard - 挑战卡片

**功能特点：**
- 🎯 挑战进度追踪
- 🏅 奖励展示
- 👥 参与者统计
- ⏰ 倒计时提醒
- 🌟 精选挑战标识
- 📊 多种状态（未开始、进行中、已完成、已失败）
- 📤 分享功能

**使用场景：**
- 运动挑战活动
- 社交健身互动
- 目标激励系统

**示例路径：** `/challenge-card`

---

## 技术实现

### 依赖
- Vue 3 Composition API
- ECharts (WorkoutZoneChart)
- CSS3 动画和渐变

### 组件导出
所有组件已添加到 `packages/src/index.js` 中，可以直接导入使用：

```javascript
import { 
  StrengthTrainingChart,
  WorkoutZoneChart,
  AchievementBadge,
  ChallengeCard
} from 'your-package-name';
```

### 示例页面
每个组件都配有完整的示例页面，展示了各种使用场景和配置选项。

## 设计理念

这4个组件的设计遵循以下原则：

1. **实用性优先** - 解决真实的运动健康场景需求
2. **视觉吸引力** - 使用渐变色、动画提升用户体验
3. **数据可视化** - 清晰展示复杂的运动数据
4. **社交互动** - 支持分享、排行等社交功能
5. **激励机制** - 通过成就、挑战激励用户坚持运动
